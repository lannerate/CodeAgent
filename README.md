# CodeAgent
Code agent for developer
## ollam install
```
wget https://github.com/ollama/ollama/releases/download/v0.1.41/ollama-linux-amd64
ollama run codestral:22b
```
## Install dependency
```
pip install -U crewai
pip install -U 'crewai[tools]'
```
## run code-agent
```
python code-agent.py
```
